# 1 Project Overview

This system is a distributed biometric attendance platform designed for educational
institutions. It uses facial embedding vectors and similarity search algorithms to
identify students quickly while preserving privacy.

The system avoids storing raw facial images and instead stores compact facial
embedding vectors generated by deep learning models.

The architecture consists of a central administrative server and multiple recognition
devices installed in classrooms or entry points.

The system is designed to operate in both online and offline modes, ensuring that
attendance recording continues even during network interruptions.

# 13 FAISS Similarity Search

FAISS index stored locally on device.

Process:

Load FAISS index  
Extract embedding from detected face  
Search nearest neighbor  
Check similarity threshold  
Return predicted identity  

Index updated whenever student embeddings change.

# 15 Offline Mode

Devices can operate without internet connection.

Workflow:

Attendance stored locally  
Device attempts periodic synchronization  
Server confirms receipt  
Local buffer cleared  

Device memory stores approximately one week of data.
